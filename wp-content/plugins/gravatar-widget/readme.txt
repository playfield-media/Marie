=== Gravatar Widget ===
Contributors: automattic, johnny5
Tags: widget, gravatar, WordPress.com
Stable tag: trunk
Requires at least: 2.8
Tested up to: 3.2

Displays a Gravatar in the sidebar of your blog. Some of the features of this plugin are:

* Email address of Gravatar (either a user on the blog, or any address)
* Size of Gravatar
* Image alignment
* URL for Gravatar (when someone clicks on the image)
* Add arbitrary text to your widget
* Fully localized

This makes it very easy to create an 'about me' widget with a short biography and a photo.

== Installation ==

You can quickly install the plugin by going to Plugins->Add New from your dashboard and typing `gravatar widget` into the search area. Full instructions will be provided.

Manual Install:

1. Download `gravatar-widget.zip`
1. Unzip
1. Upload the `gravatar-widget` directory to your `/wp-content/plugins` directory
1. Go to the plugin management page and enable the plugin
1. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.
